# 最优化学习笔记

本仓库记录《**最优化：建模、算法与理论**》（刘浩洋、户将、李勇锋、林宙辰）的学习过程，包含每章的**笔记**、**习题解答**与配套的 **Python 代码实现**。

## 📖 内容组织

每一章对应一个 `chap_N/` 目录，内部约定：

| 文件 | 说明 |
|------|------|
| `note.md` | 章节笔记：定义、定理、证明与个人理解 |
| `exercises.md` | 习题解答 |
| `*.py` | 概念/算法的代码实现与实验 |

## 🗂️ 目录结构

```
.
├── LEARNING_PLAN.md        # 学习计划：8 阶段 / 8 章的内容与重点
├── chap_1/                 # 第 1 章 · 最优化概述
│   ├── note.md
│   ├── exercises.md
│   ├── sparse_opt.py       # ℓ1 vs ℓ2 稀疏优化对比（CVXPY）
│   └── lp_ball_plot.py     # ℓp 单位球可视化
├── chap_2/                 # 第 2 章 · 数学基础
│   ├── note.md
│   ├── exercise.md
│   ├── backprop_manual.py  # 手写反向模式自动微分示例
│   └── convex_hull.py      # 凸包
├── environment.yml         # conda 环境（完整）
└── environment_light.yml   # conda 环境（精简）
```
